# ipvsadm

ip转法的的配置命令;

```
ipvsadm -A -t 192.168.143.78:31555 -s rr -p 600
ipvsadm -a -t 192.168.143.78:31555 -r 10.244.3.5:8080 -m
ipvsadm -lc
```



```
//删除一条虚拟服务器记录；
ipvsadm -D -t 192.168.143.44:5432 
```

```
#删除转发规则
ipvsadm -d -t 192.168.143.44:5432 -r 10.244.0.9:5432
```

```
#增加一条虚拟服务器， 采用tcp协议（-t),轮训算法采用wlc（默认，-s指定），持久链接（-p 超时时间）；
ipvsadm -A -t 192.168.143.44:5432 -s wlc -p 7200
```

```
#增加一条转发规则 （-m 负载均衡，-w weight)
ipvsadm -a -t 192.168.143.44:5432 -r 10.244.0.9:5432 -m  -w 1
```
